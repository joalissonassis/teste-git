lista de comandos

  • git status -
serve para verificar se tem alguma alteração do código
  • git commit -
deixa o código pronto para subir para o repositório
  • git push -
sobe o código para o repositório
  • git branch -
cria ramificação do código (como loki em vingadores/ outra linha temporal)
  • git merge -
    • faz a junção da ramificação com o código principal
  • git checkout
    • serve para criar e verificar qual branch estamos usando
